Think Tank

Coding comrades!

We know how stressful learning to code is. Come hang in the Think Tank when you need to relax and step away from syntax for a while. Remember, you are learning, and you are growing. Imagine trying to learn French, Spanish, Russian and German all at once.. That's basically what you're doing now. Each language syntax you learn is another part of the puzzle. It's okay to get stressed, but you took this path for a reason; remember that.
You can and will get better. Please don't give up!

Tips

  • If you're not making any progress, step away from your computer for a while. Sometimes all it takes to problem solve your code is a fresh pair of eyes.
  • Don't bug out when there's a bug in your code. You are still a beginner, you don't have the experience to create perfect code - because perfect code doesn't exist.
  • Talk with people who have no idea what you're talking about. Often times, simply reiterating the issue out loud in a way that's easily digestible will help more than anything.
